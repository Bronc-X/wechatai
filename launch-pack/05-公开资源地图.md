# 公开资源地图

## 微信生态底座

### CloudBase AI

用途：

- 小程序 AI 能力接入。
- Agent 对话。
- 云函数 / 云托管部署。
- 后续连接微信生态。

链接：

- 腾讯云开发 AI 指引：https://cloud.tencent.com/document/product/876/116226
- CloudBase Agent 开发：https://docs.cloudbase.net/en/ai/agent-development/
- CloudBase + LangGraph：https://docs.cloudbase.net/en/ai/agent-development/frameworks/langgraph

建议：

- MVP 可以优先用 CloudBase 做微信生态后端底座。
- 不要一开始自建 GPU。
- 模型保持可替换，核心沉淀在 schema 和业务流。

## Agent 编排

### Dify

用途：

- 快速搭建诊断报告生成、预约意图解析、垂直 demo。
- 适合销售 demo 和早期交付。

链接：

- Dify Agent Node：https://docs.dify.ai/en/use-dify/nodes/agent
- Dify Workflow API：https://docs.dify.ai/api-reference/workflows/run-workflow

### LangGraph

用途：

- 做正式产品中的可控 Agent 编排。
- 适合下单、预约、支付确认、异常恢复这类需要状态机的业务。

链接：

- LangGraph：https://www.langchain.com/langgraph
- Human-in-the-loop：https://docs.langchain.com/oss/python/langchain/human-in-the-loop

### n8n

用途：

- 连接 CRM、表格、客服、订单系统和第三方服务。
- 适合客户交付中的自动化流程。

链接：

- n8n AI Agent：https://docs.n8n.io/integrations/builtin/cluster-nodes/root-nodes/n8n-nodes-langchain.agent/
- n8n Tools Agent：https://docs.n8n.io/integrations/builtin/cluster-nodes/root-nodes/n8n-nodes-langchain.agent/tools-agent/

## 小程序 UI / 体验升级

### WeUI

用途：

- 微信官方风格组件。
- 适合保持微信原生感。

链接：

- https://developers.weixin.qq.com/miniprogram/dev/platform-capabilities/extended/weui/

### TDesign Miniprogram

用途：

- 腾讯开源小程序组件库。
- 适合搭建现代、规范、可扩展的小程序界面。

链接：

- https://github.com/tencent/tdesign-miniprogram
- https://tdesign.tencent.com/miniprogram/overview

### Vant Weapp

用途：

- 电商、本地生活、表单、列表场景。
- 轻量可靠，生态成熟。

链接：

- https://vant-ui.github.io/vant-weapp/

### Ant Design Mini

用途：

- 移动端业务系统、筛选、表单、列表。

链接：

- https://mini.ant.design/

### NutUI

用途：

- 电商、零售、商品详情、活动页。

链接：

- https://nutui.jd.com/

### Taro

用途：

- React / Vue 跨端开发微信小程序和 H5。
- 适合后续做标准化前端改造包。

链接：

- https://docs.taro.zone/docs/

### CodeFun / Pixso / 即时设计

用途：

- 设计稿到小程序代码辅助。
- UI 改版初稿和交付加速。

链接：

- CodeFun：https://code.fun/
- Pixso：https://pixso.cn/
- 即时设计：https://js.design/

建议：

- 不要完全依赖设计稿转代码工具生成生产代码。
- 更适合作为“设计草稿 -> 小程序页面初稿 -> 工程师整理”的加速链路。
